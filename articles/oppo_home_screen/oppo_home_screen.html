【OPPO】adbでホーム画面のGoogle Discoverフィードを消す方法

公開日：2026年6月5日
最終更新日：2026年6月5日

OPPOはホーム画面のGoogle Discoverフィードが設定で消せない

GMSが標準搭載されたほとんどのAndroid端末は、Google Discoverフィードというニュースや天気などが表示されるページがデフォルトでホーム画面の左端に配置されている。 これはほとんどの端末ではホーム画面の設定から非表示にできるが、OPPOやOnePlus、RealmeといったColorOS系統搭載端末では非表示にする設定が存在しない。 Googleアプリの設定でDiscoverをオフにしてニュースや天気を非表示にすることはできるが、ホーム画面からページが消えるわけではない。 Googleアプリを無効化することでも非表示にできるが、Gemini、Googleレンズ、音声検索といった、Googleに関連する一部の機能が使えなくなってしまう。 しかし、非表示の設定が表示されなくても設定自体は存在するので、adbコマンドで非表示にすることは可能である。 ここではその方法を解説する。

手順

  • 開発者オプションを有効にする
  • adbで接続する
  • adbコマンドを実行する

テスト環境

以下ではOPPO Find X9 Ultraを用いてテストを行っている。

OPPO Find X9
Amazonで開く

開発者オプションを有効にする

OPPOにadbで接続できるようにするために、開発者オプションを有効にする必要がある。 この手順はadbをOPPOで実行する場合もPCで実行する場合も同様である。

方法

1.設定アプリを開き、 "デバイスについて" をタップする

2. "バージョン" をタップする

3. "バージョン番号" を7回タップし、パスワードを入力する

これにより、開発者オプションが有効になる。

adbで接続する

adbのワイヤレスデバッグを用いて、OPPOからそのスマホ自体にadbコマンドが実行できるようにする。 adbコマンドをPCで実行する場合はOPPOで設定アプリを開き、 "システムおよび更新" の "開発者オプション" の "USBデバッグ" をオンにして "USB デバッグを許可しますか?" に対して "OK" をタップし、PCとOPPOをUSBケーブルで接続して "USB デバッグを許可しますか?" に対して "OK" をタップする。

方法

1.ADB Shellをインストールする

以下のリンクからADB Shellをインストールする

2.ADB Shellを開き、"はい" をタップする

3. "接続方法を選択" で "Shizuku(Android 11+)" をタップする

4. "接続" をタップする

5. "はい" をタップする

6. "接続" をタップする

7. "はい" をタップする

8. "許可" をタップする

9. "接続" をタップする

10. "はい" をタップする

ここで設定アプリの開発者向けオプションが開かれる。

11. "USBデバッグ" をオンにする

12. "OK" をタップする

13. "ワイヤレスデバッグ" をオンにする

14. "許可" をタップする

15. "ワイヤレスデバッグ" をタップする

16. "ペア設定コードによるデバイスのペア設定" をタップする

17.通知の "ペアリングコードを入力するにはクリック..." をタップする

18.Wi-Fiペア設定コードを通知に入力して送信する

成功すると "ペアリング成功" と通知が来る。

19.ADB Shellを開き、 "はい" をタップする

20. "ターミナル" タブの "ローカルターミナル" をタップする

adbコマンドを実行する

ホーム画面の左端のページを非表示にするadbコマンドを実行する。 この手順はadbをOPPOで実行する場合もPCで実行する場合も同様である。

方法

1.ホーム画面の左端のページを非表示にするadbコマンドを実行する

以下のコマンドを両方実行する。

adb shell settings put secure assistant_screen_type 0
adb shell settings put secure assistant_screen_type_left_enable 0

2.ランチャーアプリを強制終了する

設定を反映するため、ランチャーの再起動が必要になる。 設定アプリを開き、 "アプリ" の "アプリ管理" を開いて "システムランチャー" をタップし、 "強制停止" をタップしてもいいし、端末を再起動してもいいが、以下のadbコマンドで強制終了したほうが速い。 これにより、画像2枚目から3枚目の変化のようにホーム画面の左端のGoogle Discoverフィードが非表示になる。 

adb shell am force-stop com.android.launcher

もとに戻したくなった場合、以下のadbコマンドを両方実行する。

adb shell settings put secure assistant_screen_type 2
adb shell settings put secure assistant_screen_type_left_enable 1

メリット

･Googleアプリを無効化せずにGoogle Discoverフィードをオフにできる

GeminiやGoogle レンズ、音声検索がそのまま利用できる。

デメリット

･設定の変更に手間がかかる

設定を変更する際、毎回adbコマンドを実行する必要がある。

ColorOSのadbコマンド事情

今回のadbコマンドは他のスマホと同じように実行できたが、ColorOSの端末では一部のadbコマンドはシステム最適化を無効にしないと実行できない。 これはプリインストールアプリの無効化やアプリへの権限の付与などが対象であり、システム最適化が有効の状態で実行するとエラーになる。 システム最適化は開発者オプションから無効にできるが、2026年6月現在、ColorOS 16.07以降でシステムの言語を英語にしないとこの設定項目が表示されないバグが発生している。 また話は変わるが、過去には国内版のOPPO端末で再起動を連続で繰り返すとカメラアプリにシャッター音の項目が現れるというバグが発生していた。 ColorOSはこのように設定項目の表示に関わるバグが多いのかもしれない。
